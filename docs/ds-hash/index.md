# 数据结构（四）：哈希表


哈希表是针对查询，空间换时间的一种数据结构，一般能够在 O(1) 时间内插入、查询、删除元素，
但不能通过下标访问元素。

哈希表分为哈希集合和哈希映射，哈希集合存储非重复值，哈希映射存储 key 唯一的 (key,value) 对。

> python 中一般直接用内置库 set 和 dict 表示哈希集合和哈希映射。
> 另外还有一些特定用途的内置库，比如 Counter, defaultdict, OrderedDict。
