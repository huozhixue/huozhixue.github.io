# 人类简史


> <u>**人类简史**</u>，又名：Sapiens: A Brief History of Humankind

## 简介

《人类简史：从动物到上帝》是以色列历史学家尤瓦尔·赫拉利创作的历史类著作，2012年首次出版，林俊宏翻译。作者为牛津大学历史学博士，现任耶路撒冷希伯来大学历史系教授，研究领域涵盖世界历史与宏观历史进程 。该书2015年3月进入《纽约时报》畅销书榜，次月获得第十届文津图书奖。

著作以演化生物学为基础，将人类历史划分为认知革命、农业革命、人类的融合统一与科学革命四个阶段。认知革命使智人通过虚构故事实现大规模协作，农业革命导致人类生活方式改变并引发社会阶层分化，科学革命则以承认无知的创新观念推动科技发展 。书中同时探讨了帝国兴衰、宗教法则、资本主义教条等社会建构的虚构本质，并对人类幸福状态与环境代价提出反思。全书融合多学科视角，涉及物理学、化学、生物学、生态学和政治学等领域。
