# 历史的尘埃


## 简介

《历史的尘埃》是已完结于起点中文网的奇幻类网络小说，作者是知秋。 该小说以电脑游戏《英雄无敌3》作为全书框架和设定的基础，构建了一个包含精灵、人类、亚人类等多个种族的世界，充满了剑与魔法的奇幻色彩的故事。

在历史的天空下，不管你爱与不爱都只是其中的尘埃。无论是谁，所能把握的不过也只是自己而已。
