# 算法（一）：位运算与进位制


位运算是计算机最基础的运算，包括位模式或二进制数的一元和二元操作（&、|、^、~、<<、>>）。

很多位运算的问题具有较强的技巧性，需要很熟悉各个位运算符的性质。
