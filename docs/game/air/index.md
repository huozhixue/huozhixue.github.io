# air


## 简介

《Air》是日本Key会社继《Kanon》后，所出品的第二款恋爱AVG（电子小说类游戏）游戏。

AIR的标志是一个牵着男孩的手的女孩。故事主要以夏天为主基调，讲述了一个跨越千年的翼人传说。
亦被视为Key社季节组曲中的“夏”。


**制作人员：**
- 开发：Key
- 剧本：云龙寺魁、丘野塔也、麻枝准、凉元悠一
- 音乐：折户伸治、戸越まごめ
