# island


## 简介

《ISLAND》是Frontwing制作的冒险电脑游戏，Frontwing15周年纪念作品。
故事穿插着各种各样的关键词，通过巧妙的伏笔， “世界的秘密”一点点揭晓。

**制作人员：**
- 开发：前翼
- 剧本：G.O.
- 音乐：安濑圣
- 人物设定：空中幼彩
- 主题歌作曲：藤田淳平
- 主题歌演出：riya
