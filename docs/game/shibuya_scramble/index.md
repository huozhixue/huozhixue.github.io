# 428 ～被封锁的涩谷～


## 简介

《428 ～被封锁的涩谷～》是日本Chunsoft公司开发的游戏。该作于2008年12月4日以Wii为平台正式发售。
该作故事情节围绕一起发生在日本东京涩谷地区的绑架案展开。
2017年，发行商Spike Chunsoft正式宣布，该作于2018年登陆PC，PS4平台。

**制作人员：**
- 开发：CHUNSOFT
- 发行：Spike、世嘉
- 剧本：奈须蘑菇、北岛行德、我孙子武丸
- 音乐：佐藤直纪
- 人物设定：武内崇
- 导演：石井次郎
