# 时空幻境


## 简介

《时空幻境》是由Number None开发，微软游戏工作室及Number None发行的一款动作类解谜游戏。于2008年8月6日发布，游戏亦于2009年登陆PC、Mac、PS3平台。
 
《时空幻境》允许玩家随时使用时间倒退功能让主角回到以前的位置。玩家需要灵活运用时间倒退功能以及各种加快、减慢甚至倒退时间的机关，收集散落在关卡角落的拼图碎片，方可完成游戏。

**制作人员：**
- 遊戲設計師：Jonathan Blow
