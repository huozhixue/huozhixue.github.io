# ever17


## 简介

infinity系列第二部。本游戏分为两个主人公视点——仓成武和失忆少年，与小町月海，茜崎空，优， 松永沙罗，
八神可可几位少女编制一团迷离的故事，各女主角的剧情安排也因主人公的视点而有所变动未来海洋主题乐园
内演绎生离死别的悲喜剧。


**制作人员：**
- 开发：KID
- 发行：光谱资讯、赛科、サイバーフロント、北京娱乐通科技发展有限公司
- 剧本：打越钢太郎、中泽工、梅田伸明
