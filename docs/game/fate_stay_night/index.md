# Fate/stay night


## 简介

《Fate/stay night》是TYPE-MOON开发并最早于2004年1月30日在PC平台上发售的文字冒险游戏，也是TYPE-MOON公司商业化后初次亮相的作品，是Fate的系列作品之一。

**制作人员：**
- 开发：型月
- 发行：Notes.
- 剧本：奈须蘑菇
- 音乐：芳贺敬太、永田大祐、石川敏
- 主题歌作曲：NUMBER 201
- 主题歌演出：M.H.
- 原画：武内崇
