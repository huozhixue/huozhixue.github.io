# eden*


## 简介

《eden*》是由日本游戏公司minori于2009年所推出的一款恋爱冒险游戏。

该作品是一款以恋爱为题材，架设在虚幻未来为背景下的交互式小说。eden*是minori的第6部作品，
也是在继取得成功的《ef -a fairy tale of the two》（悠久之翼）之后的作品。

**制作人员：**
- 开发：minori
- 剧本：镜游
- 美工：结城辰也
- 音乐：柳英一郎、天门
- 主题歌作词：酒井伸和
- 主题歌演出：原田瞳
- 原画：ちこたむ
