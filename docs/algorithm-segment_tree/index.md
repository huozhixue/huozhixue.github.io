# 力扣总结 数据结构进阶（六）：线段树


线段树（Segment Tree）是一种二叉树形数据结构。多用于区间查询。

相比于 [前缀和](/algorithm-prefix_sum) 和
[树状数组](/algorithm-binary_indexed_tree)，
线段树更复杂，也更通用。

> 线段树可以区间修改、区间查询。而且线段树不仅能维护区间的和，还可以维护区间的
>最小值、最大值、总和、最大公约数、最小公倍数等。
