# 0371：两整数之和（★★）



## 题目

给你两个整数 a 和 b ，不使用 运算符 + 和 - ，计算并返回两整数之和。

示例 1：
    
    输入：a = 1, b = 2
    输出：3

示例 2：

    输入：a = 2, b = 3
    输出：5
 

提示：
- 1000 <= a, b <= 1000


## 分析

可以用位运算来解决，但 python 处理本题的负数较为麻烦，所以直接调库。

```python
def getSum(self, a: int, b: int) -> int:
	return sum([a,b])
```
