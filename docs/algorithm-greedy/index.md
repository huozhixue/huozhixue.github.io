# 力扣总结 常见算法（八）：贪心


贪心算法是一种高效的搜索方法。在每一步，都选择当前最好的分支，最终即能得到最优解。

能用贪心算法解决的问题，对于贪心正确性的证明往往更加复杂。
因此很多时候需要依赖一定的直觉才能想到贪心策略，即先构造后证明。

> 而有些不能用贪心解决的问题，如果对用例的考虑不够周全，容易错误地采用贪心策略。
因此使用贪心一定要慎重。
