# 力扣总结 算法进阶（一）：前缀和


前缀和是一种常用的解决区间查询问题的算法。

当数组固定时，根据前缀和即可在 O(1) 时间查询任意区间的和。

> 当数组动态变化时，则考虑用 [树状数组](/algorithm-binary_indexed_tree)、
>[线段树](/algorithm-segment_tree) 等方法
