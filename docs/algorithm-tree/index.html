树是一种非线性的数据结构，本质是节点的有限集。其定义是递归的：

  • 有且仅有一个特定的称为根（root）的节点
  • 当节点数量 > 1 时，其余节点可分为若干个互不相交的有限集，其中每个集合也可以看作一颗树，称之为根的子树。

从图的观点来看，树也可视为一个拥有 N 个节点和 N-1 条边的一个有向无环图。

树的应用非常广泛，也常和其它数据结构和算法有联系，尤其是递归。
