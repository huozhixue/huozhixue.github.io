# 蓝色恐惧


## 简介

雾越未麻是流行音乐偶像团体“CHAM!”中的成员之一，但是在公司的决策下，她开始转型为演员。但她却感觉内心有个声音在拒绝自己的选择，而未麻的一些歌迷也反对这个决定。随着越来越多电视剧的演出，未麻却觉得自己的头脑越来越奇怪，仿佛有“另一个自己”正在形成。这时，她身边的工作人员竟一个个接连被杀。在面对社会压力和疑云之际，未麻感到越来越混乱…

**制作人员：**
- 原作：竹内义和
- 导演：今敏
- 脚本：村井贞之
- 演出：松尾衡
- 音乐：几见雅博
- 人物原案：江口寿史
- 人物设定：滨洲英喜
