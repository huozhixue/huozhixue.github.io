# 纯洁的玛利亚


> <u>**[纯洁的玛利亚](https://bgm.tv/subject/84801)**</u>，又名：純潔のマリア、Junketsu no Maria

## 简介

以中世纪的法兰西王国为舞台，讲述了在百年战争的时代，身为最强魔女同时也是圣女的玛利亚十分厌恶战争，她拥有的强大魔力足以让纷争消失。然而，如果她失去了自己的处女之身，那么届时也将失去强大的力量。
