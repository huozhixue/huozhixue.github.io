# 缘之空


## 简介

远离都市的乡间、奥木染。春日野悠带着他的妹妹·春日野穹来到了这里。
兄妹俩幼时暑假期间曾来过多次奥木染，这里有着祖父的家，是个令人怀念的地方。
因为意外事故而失去了双亲与去处的兄妹，决定移居至现在没有人居住的祖父家，在此开始新的生活。

觉得自小以来就没怎么变化的街道与居民、教人怀念的回忆与静谧的环境，让悠倍感舒适。但徐徐地，变化来到了。
阳光日渐变得强烈的初夏天空之下，故事开始了。
