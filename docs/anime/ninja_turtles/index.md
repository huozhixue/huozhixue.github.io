## 简介

《忍者神龟》讲述了在纽约市一条大街的地下管道里住着四只功夫高强的忍者神龟和他们的老师斯普林特—一只来自日本的超级大老鼠。神龟们各个骁勇善战，身怀绝技，为了维护正义，保卫城市家园，他们同狡猾的巴克斯特·斯多克曼博士、紫龙帮的坏蛋以及一伙神秘且训练有素的忍者（史莱德的部下）展开殊死搏斗……
