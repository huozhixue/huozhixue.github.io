# 魔法少女什么的已经够了啦。


## 简介

被垃圾堆里的奇妙生物说了“你有魔法少女的素质”的柚香。
变身成魔法少女的她，衣服竟然是泳装！？而且也没有什么敌人，不去帮助别人、不去收集道具也无所谓！？
漫无目的的魔法少女生活开始了……。
