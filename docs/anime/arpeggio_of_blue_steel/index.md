# 苍蓝钢铁的琶音


## 简介

在21世纪初，人类因全球暖化的影响而失去了大半的陆地。此时，在世界各地突然出现了搭载着神秘超兵器被称为“雾之舰队”的战舰群，对其毫无招架之力的人类失去了制海权，并且再也无法自由航行于大海之上。17年后，千早群像与其同伴们在不知觉中搭上了潜水舰伊401，并与伊401的心智范本伊欧娜一起挑起了与雾之舰队间的战火。

**制作人员：**
- 原作：Ark Performance
- 导演：岸诚二
- 脚本：森田繁、中村浩二郎、上江洲诚、日暮茶坊
- 分镜：吉冈忍、高桥正典、政木伸一
