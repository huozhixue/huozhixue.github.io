## 简介

I-MACHINE——这是在极限环境之中进行作业的大型机器人的总称。
联盟学院的学生·三栗麻耶在使用I-MACHINE时被卷入了一场事故，幸而被同样使用I-MACHINE来采掘“欧利哈尔特矿石”的一批人救下，并坐上了他们的宇宙船。
就这样成为了实际上等同于宇宙海盗的埃斯卡维特公司的“职员”之一……。
