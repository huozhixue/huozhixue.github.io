# 棺姬嘉依卡


## 简介

于战乱后的菲尔毕斯特大陆流浪的乱破师——托鲁·亚裘拉，因为失去了乱破师的生存目的，而每天过着庸庸碌碌，并被妹妹阿卡莉责备的生活。某日，在托鲁前往山中的时候，与谜之少女嘉依卡相遇了……

**制作人员：**
- 原作：榊一郎
- 导演：增井壮一
- 脚本：土屋理敬、待田堂子
- 分镜：塚田拓郎、小林孝志、大桥誉志光、寺冈岩
