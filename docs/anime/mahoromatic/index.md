# 魔力女管家


## 简介

为了和外星侵略者进行抗争，组织“太白星”制造了最强战斗用人型机器人麻幌。在她的生命即将终结之际她离开了战场，将自己剩余的时间投入到少年美里优的女仆工作之中。等待她的会是平静的生活吗……

**制作人员：**
- 原作：Ditama某、中山文十郎
- 导演：山贺博之
- 脚本：佐伯昭志
- 分镜：山口赖房、高桥拓朗、铃木利正
