# 海月姬


> <u>**[海月姬](https://bgm.tv/subject/6811)**</u>，又名：海月姫

## 简介

仓下月海是一名梦想着成为插画家，并且非常喜欢水母的女孩。她为了梦想而来到东京，后入住了女生公寓“天水馆”，这里的住客大多是宅女，对她们来讲拥有时尚感的人就全是她们的敌人。一天，为了营救水母，月海遇到了一名超美的伪娘。于是发生在宅女、伪娘之间的故事就此展开。
