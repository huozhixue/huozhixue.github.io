# ReLIFE


## 简介

海崎新太（27岁）在作为新毕业生进入的公司工作了3个月就辞职了。
之后他的就职活动也不顺利。
双亲寄来的生活费也中断了，不得已只好回到乡下。
能够倾听他烦恼的朋友和女友全都没有……
穷途末路的海崎面前出现了一位神秘人物·夜明了。
夜明向海崎提出，要他参加以尼特为对象的社会复归程序“ReLIFE”。
其内容是，利用神秘的秘药，仅仅让外表回复年轻，并在一年内以高中生的身份去读高中——。

**制作人员：**
- 原作：夜宵草
- 导演：小坂知
- 脚本：兵头一步、横手美智子
- 分镜：备前克彦、关晓子、山田弘和、添田和弘
