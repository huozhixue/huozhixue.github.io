# 川之光


> <u>**[川之光](https://bgm.tv/subject/12436)**</u>，又名：川の光

## 简介

故事叙述父子三鼠住在河边过着平静的生活，原以为能在河畔眺望河光度过一生，死后化成雨、汇成小流，与喜爱的河川永远融合一体。不料，东京都建设局打算兴建工程将河川铺成暗渠，它们只能离开家园，展开漫长惊险的旅程。
