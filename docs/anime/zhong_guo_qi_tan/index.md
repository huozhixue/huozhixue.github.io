# 中国奇谭


> <u>**[中国奇谭](https://bgm.tv/subject/358660)**</u>，又名：Yao-Chinese Folktales

## 简介

动画短片集《中国奇谭》由八个植根于中国传统文化的独立的故事组成：《小妖怪的夏天》、《鹅鹅鹅》、《林林》、《乡村巴士带走了王孩儿和神仙》、《小满》、《玉兔》、《小卖部》、《飞鸟与鱼》。故事纵览古今、展望未来，从乡土眷恋到科技幻想、从生命母题到人性思考，展现着中式想象力、承载着中国民族文化与哲学。
