# 藤本树17-26


> <u>**[藤本树17-26](https://bgm.tv/subject/582501)**</u>，又名：藤本タツキ17-26、Fujimoto Tatsuki 17-26

## 简介

以《炎拳》、《链锯人》和《Look Back -蓦然回首-》响乎盛名的鬼才漫画家藤本树，于其17至26岁期间所创作的八部短篇作品生动地描绘了青春爱恋、混乱、疯狂，以及人际之间的情感纠葛，如今这些作品全部动画化！

PART.1聚焦于《17-21》的动画化合集，内容涵盖了人类灭亡后最后两位生还者之间的羁绊的《从前，庭院里有两只鸡。》、描写少年爆炸冲动的《佐佐木同学接住了子弹》，宇宙级失控的科幻爱情喜剧《恋爱是盲目的》以及疯狂少女杀手的爱情故事《Shikaku》。

PART.2聚焦于《22-26》系列，包括以海中钢琴谱写少年与人鱼浪漫恋曲的《人鱼狂想曲》、探讨性别框架与自我认同的《一觉醒来变成了女孩子的病》、讲述宿命兄妹重生的殒地悲歌《预言的那由多》以及描绘姐妹间嫉妒与成长的《妹妹的姐姐》。

短篇集由六家动画工作室联手七位导演精心打造，以各自独特的视觉风格，重新诠释藤本树天马行空、绚丽多姿的创作世界！
