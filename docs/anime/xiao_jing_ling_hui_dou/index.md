# 小精灵灰豆


## 简介

小精灵灰豆儿是一个长得很丑而心地却很善良的小妖精，立志要去掉身上的妖精影子，做一个真正的人。在神通广大又爱胡闹的八戒、大圣的帮助下，在海底龙宫打跑了祸害人间的大嘴怪；好心为一条龙画上眼睛，却把人家眼睛给占歪了；把红色鱼鳞物归原主；在弥勒佛那里练成了铁屁股的本事，勇敢地去补天—尽管小精灵灰豆儿受尽了委屈，但他还是想成为一个好人。
