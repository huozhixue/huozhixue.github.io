# 火球


## 简介

在遥远的未来，一颗普通的行星上——
由于人类的抵抗，机器人的贵族政治正走向灭亡。
在与世隔绝的住宅中，两位机器人——Flügel公爵的女儿Drossel，及其管家Gedächtnis，展开了琐碎的“日常”对话。
