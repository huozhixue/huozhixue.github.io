# 神的记事本


> <u>**[神的记事本](https://bgm.tv/subject/15237)**</u>，又名：神様のメモ帳

## 简介

舞台是都市市中心附近的地方。描寫著普通的高中生藤島鳴海與僱主偵探愛麗絲、與她的助手尼特族們活躍的故事。故事當中有些描寫了現實中暴力集團、與毒品等事件。同時也是描寫尼特族青少年有點不堪、有點可笑，又帶著一絲淡淡哀愁的青春冒險故事。
