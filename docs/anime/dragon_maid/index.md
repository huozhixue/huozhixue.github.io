# 小林家的龙女仆


## 简介

在单身的辛苦OL小林身边突然出现的女仆装束的美少女托尔。
长着角和尾巴的她的身姿正是所谓的龙娘。
在醉酒的小林邀请下说要到家里去的托尔，鬼使神差地开始以小林家女仆的身份工作……！？
“女仆”+“龙”=“女仆龙”有着笨手笨脚的可爱之处！
龙娘与人类之间基本上很温暖、偶尔有些黑暗的异种族间交流喜剧！！
