# 调酒师


## 简介

酒吧不仅仅是一个喝酒的地方，在调酒师高超的技艺下，调出的不仅仅是美味的鸡尾酒，还有着百味的人生。本作可以说为奔忙于世的各种颜色的“领”们量身定做：那么，今晚想要怎样的一杯呢？在你说出口之前，已为你奉上最佳的一杯，这才是调酒师的风采。

**制作人员：**
- 原作：城亚罗树
- 导演：渡边正树
- 脚本：今川泰宏
- 分镜：黑川文男、小林一三、三原武宪、康村谅
- 演出：冈崎幸男
