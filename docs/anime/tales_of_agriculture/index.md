# 萌菌物语


## 简介

拥有用肉眼看见“菌”能力的少年泽木直保和青梅竹马结城萤进入农业大学所发生一连串的趣事，透过直保和其他人物的互动可以了解许多跟菌相关的知识，也因为直保的特殊能力而招惹许多事情……

**制作人员：**
- 原作：石川雅之
- 导演：矢野雄一郎
- 脚本：猪爪慎一、武上纯希、高桥奈津子
- 分镜：田头忍、镝木宏、渡边了
