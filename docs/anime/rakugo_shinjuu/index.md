# 昭和元禄落语心中


## 简介

无法忘记在刑务所的落语慰问会中见到的大名人·八云的“死神”，出狱的与太郎径直前往寄席。拜倒在八云面前，成为他住所雇工以及弟子的与太郎，却发现在八云之处住着一位名叫小夏的女性。八云与小夏似乎有着他人无从触及的因缘……。

**制作人员：**
- 原作：云田晴子
- 导演：小俣真一
- 脚本：中西泰大、熊谷纯、待田堂子、柿原优子
- 分镜：园田雅裕、阿部达也
