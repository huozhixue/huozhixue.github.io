# 夏娃的时间


## 简介

夏娃的时间（日语：イヴの時間）是2008年8月的一部科幻在线动画。讲述了未来的日本，机器人十分普及的情况。主人公陆夫偶然间发现了一间咖啡店，在里面人与机器人是平等的。
（Via wikipedia）

**制作人员：**
- 原作：吉浦康裕
- 人物设定：茶山隆介
- 美术监督：安喰秀一
- 色彩设计：井上あきこ
