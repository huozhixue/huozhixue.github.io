# 人鱼又上钩


## 简介

以说博多腔的人鱼室见小姐以及常钓到室见的少年向岛拓郎为中心，描写人类与传说生物之间的搞笑互动。

**制作人员：**
- 原作：名岛启二
- 导演：吉原达矢
- 脚本：笔安一幸、凉村千夏
- 分镜：濑藤健嗣、中村宪由、中野英明
- 演出：野木森达哉
