# 起风了


> <u>**[起风了](https://bgm.tv/subject/56846)**</u>，又名：風立ちぬ、The Wind Rises、风起了、风起、风立、风儿起、风雪黄昏

## 简介

剧情为混合改编了零式舰上战斗机设计者堀越二郎的故事、以及小说家堀辰雄作品《风吹了》的内容，并预计2013年夏季上映。另外，男主角堀越二郎的配音，则由《新世纪福音战士》的导演庵野秀明出任。
