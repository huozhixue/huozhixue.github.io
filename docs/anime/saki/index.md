# 天才麻将少女


## 简介

天才麻将少女宫永咲虽然还是一位刚升入高中一年级的学生，但是便能以正负0分的方式打完牌局从而创下赫赫威名。故事二号女主角原村和则是在全国中学生麻将大赛个人组中夺得冠军的少女，两人因缘而聚到了麻将部，并以高中校际比赛的顶点为目标展开一番自摸放炮的故事。

**制作人员：**
- 原作：小林立
- 导演：小野学
- 脚本：浦畑达彦
- 分镜：木村宽、濑藤健嗣、山本珠代、小泷礼、熊泽佑嗣
