# 告诉我！辣妹子酱


## 简介

外表不管哪里不管怎么看都是金发辣妹，实际却是纯情女生。以有着辣妹外表和直率温柔的内心这种反差萌的辣妹子为中心，戴着眼镜小巧玲珑的宅子，以及举止文雅天真烂漫的大小姐，三个人在各种场所展开女生会话的日常喜剧。

**制作人员：**

- 导演：川口敬一郎
- 演出：藤井隆文、新田义方、松下周平、冈村正弘、关大
