# 信长的忍者


## 简介

1555年，时为战国时代。
有位青年怀抱着远大的梦想，被人称作大傻瓜的他正是后来的织田信长。
有一位仰慕着青年的少女，被信长的梦想所吸引的她笑着说道“我要成为信长大人的忍者！”
目标只有一个，那就是天下布武！

**制作人员：**
- 原作：重野なおき
- 导演：大地丙太郎
- 演出：山口光、神原敏昭、备前克彦、赤城博昭、菅井嘉浩
