# 瑞克和莫蒂


## 简介

天才兼疯子科学家Rick在失踪多年后突然回到女儿Beth的身边，并且在她的车库里搞了一个科学实验室。Rick有一把“传送门枪”，可以穿越到宇宙的各个次元。孙子Morty莫名其妙就成了他的助手，经常被拖进他的自制太空船内，跟他一同开展各种疯狂刺激的宇宙冒险。女儿女婿对他的疯狂行为感到不满，却又对这个天才科学家无可奈何。

**制作人员：**
- 导演：丹 哈蒙、贾斯汀·罗伊兰
- 脚本：麦克·麦克马汉、汤姆·考夫曼、瑞安·里德利
- 演出：杰夫·迈尔斯、约翰·赖斯
