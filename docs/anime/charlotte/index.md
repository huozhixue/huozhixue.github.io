# 夏洛特


> <u>**[夏洛特](https://bgm.tv/subject/120925)**</u>，又名：Charlotte、シャーロット

## 简介

思春期的少年少女中极少部分人会发作的特殊能力。
驱使着无人知晓的能力，过着顺风顺水学园生活的乙坂有宇。
突然出现在这样的他的面前的少女，友利奈绪。
从与她相遇之日起，被揭露出来的特殊能力者的宿命。
这是由麻枝准所描绘的，特殊能力者们奔跑着的青春物语——。
