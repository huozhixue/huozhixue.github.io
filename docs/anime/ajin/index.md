# 亚人


## 简介

17年前，非洲的战场上出现了绝对不会死的人，人们将这种未知的新生物称为“亚人”。在暑假即将到来之时，日本普通高中生永井圭在放学途中遭遇了交通事故而当场死亡，不过又随即复活，正因如此，其“亚人”的身份暴露，并被投以高额的悬赏。就这样，身为众人类敌人的少年逃避之旅开始了……

**制作人员：**
- 原作：樱井画门
- 导演：安藤裕章
- 脚本：猪原健太、熊谷纯、濑古浩司
- 分镜：井出惠介、鹿住朗生、福岛宏之、森田宏幸
