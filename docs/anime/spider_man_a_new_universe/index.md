# 蜘蛛侠：平行宇宙


## 简介

蜘蛛侠不止一个！漫威首部超英动画巨制《蜘蛛侠：平行宇宙》将经典漫画与CGI技术完美呈现，讲述了普通高中生迈尔斯·莫拉斯如何师从蜘蛛侠彼得·帕克，成长为新一代超级英雄的故事。影片中迈尔斯和从其它平行宇宙中穿越而来的彼得、女蜘蛛侠格温、暗影蜘蛛侠、潘妮·帕克和蜘猪侠集结成团，六位蜘蛛侠首次同框大银幕，对抗蜘蛛侠宇宙最强反派。

**制作人员：**
- 原作：斯坦·李、漫威漫画
- 人物设定：金时允（音译）
- 製作：漫威娱乐、哥伦比亚影业
- 特效：索尼图形图像运作公司
