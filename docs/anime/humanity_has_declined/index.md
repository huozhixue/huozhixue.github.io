# 人类衰退之后


## 简介

世界的人口因为不明原因而急降，逐渐步出历史的舞台，取而代之的是一种平均身高只有10公分、三头身的妖精。妖精的平均身高只有10公分，三头身的妖精们却拥有高度的智慧，而且非常喜欢吃糖果点心。主人公作为最后一届的毕业生毕业之后就回到故乡樟树之里去接续祖父的工作，成为负责斡旋在妖精与人类之间的重要国际公务员“调停官”。虽然如此，由于妖精很少跟人类接触，因此她根本没啥工作可作，因此她想要好好研究妖精的生态。
