# 乒乓


## 简介

星野裕（阿扁）是乒乓球天才，月本诚（笑匠）则是极具天赋的乒乓球好手。他们俩从小一起长大，同为片濑高中乒乓球队队员。在高校联赛中，遇上来自中国上海的孔文革、上届冠军风间龙一，关于乒乓联赛的成长的故事拉开序幕……

**制作人员：**
- 原作：松本大洋
- 导演：汤浅政明
- 演出：上野史博、伊藤良太、伊藤秀树、久保田雄大
