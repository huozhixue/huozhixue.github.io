# 黑岩射手


## 简介

近乎现实的世界中，有一位看似平凡的少女黑衣麻陶，于新一年展开其丰富璀璨的中学生活。这时，一位名为小鸟游优美的同班同学吸引了她的注意。起初对方不欲作深层交往，后来却因为彩虹小鸟的话题，而被麻陶见到小鸟游受到委屈的一面……
充满怪异的世界中，一群人生活于如眼泪般蔚蓝的天空之下，每天都仿如战士般进行着明争暗斗，仿如正为他人承受遇到的苦恼和混乱不堪的思绪……
