夏日重现，又名：サマータイムレンダ、Summer Time Rendering、夏日时光

简介

“潮死了——” 听闻青梅竹马・小舟潮讣告的网代慎平，时隔2年回到了故乡和歌山市・日都岛。

家人和朋友再会，葬礼也顺利进行。然而，好友菱形窗却告诉慎平“潮的死存在可疑之处，有他杀的可能性”。 第二天又发生了邻居一家突然全部消失的事件。随着时间的推移，慎平又听到了一个不祥的传闻。 “看到与自己一样的影子的人会死。会被影子杀死——”并且，潮的妹妹・澪说，“姐姐去世前3天看到了影子”……

在纪淡海峡浮现的夏季小岛上，穿越时空的SF悬疑片，现在拉开帷幕——
