# 落叶


> <u>**[落叶](http://bgm.tv/subject/8461)**</u>，又名：DEAD LEAVES、デッド・リーブス

## 简介


時は未来，辺り一面の荒野の中，たたずむ裸の男と女．お互い記憶も名前も覚えていなかった．
遠くに巨大な都市が見える．その中の人々は，皆同じ顔をしていた．
突然吹き飛ぶ人々．その轟音の中に完全武装したPANDYとRETROがいた．
車を奪い，縦横無尽に逃げまくる！
追いかける警官たち，壮絶なカーチェイス！！
しかし捕まり，2人は刑務所へと送られる．
その刑務所とは，二度と出られぬ月面の「DEAD LEAVES」であった．
そこで，彼女らを待ち受けるものとは一体……
