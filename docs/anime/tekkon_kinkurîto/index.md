# 恶童


## 简介

故事在“地狱之街”宝町，那里充满义气、人情和暴力。在宝町住了两个自由自在的少年，“黑”和“白”。不过，这个城市将会被重新开发，实行“孩子之城”计划，意图控制整个城市，“黑”和“白”两人的命运亦受到重大改变……

**制作人员：**
- 原作：松本大洋
- 导演：迈克尔・阿里亚斯
- 脚本：安东尼·温特劳布
- 分镜：久保正彦、西见祥示郎、森本晃司、安藤裕章
