# 18if


## 简介

这里是梦世界。
是欲望、希望、绝望成形的，意识与无意识的夹缝间。
在总是做梦的自己的房间里，再次陷入睡眠的月城遥人，偶然进入了他人的梦境。那是，心灵被囚禁在这梦世界里的魔女们的梦。陷入拒绝现实、逃往梦境、持续睡眠的“睡美人病”当中的，她们的梦的姿态。
迷失在魔女的梦世界中的遥人，寻找着出口的门，并和她们打着招呼。
