# 银河天使


## 简介

“银河天使队”是特兰斯巴尔皇国近卫军所属的特殊部队。任务是收集古代文明遗留下的“遗失科技”。但是实际上，从猫的搜索、VIP和犯罪者的护送、垃圾处理到钉在行星上的钉子的拔除，可谓是无所不做的万事屋。

**制作人员：**
- 原作：Broccoli
- 导演：浅香守生、大桥誉志光
- 脚本：佐藤胜一、泷晃一、菅良幸、杉原研二、金卷兼一、远藤明范
