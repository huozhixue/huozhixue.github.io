# 动画短片五十佳

> **1994年imdb评选的50部短篇动画作品**
## 简介

从 1914   [恐龙葛蒂](https://bgm.tv/subject/80197) 到 1988  [The Cat Came Back](https://bgm.tv/subject/318263)

恐龙葛蒂是世界上第一部角色动画。

因为拍摄于1914年，本片今天看来稚气十足，不过一只叫Gertie的雌性恐龙按字幕所示（在相关工作人员的提点下）做出一个个动作，期间她乐意有时不乐意有时。但影片当年曾让美 国观众大为吃惊，其时动画仍是新事物，真人与动画相结合的构想亦是超前。论说动画片的先驱，该片导演Winsor McCay当之无愧是其中之一。

**制作人员：**
- 导演：[Winsor McCay](https://bgm.tv/person/2615)
