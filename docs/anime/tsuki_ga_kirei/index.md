# 月色真美


## 简介

茜与小太郎。
初三时第一次被分到同班，相识的两人。
同班同学、社团伙伴、教师、父母……与周围的关联，自己的成长。
在那个一边被变化与不安追着，一边慌慌张张地向前飞奔的季节之中，稚嫩而又令人眩目的青春期之恋。

**制作人员：**
- 导演：岸诚二
- 脚本：柿原优子
- 分镜：及川启、岩崎光洋、浅井义之、平井义通、斋藤德明、平峰义大
