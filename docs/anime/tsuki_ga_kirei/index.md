# 月色真美


> <u>**[月色真美](https://bgm.tv/subject/207573)**</u>，又名：月がきれい、as the moon, so beautiful.、Tsukigakirei、Tsuki ga Kirei

## 简介

茜与小太郎。
初三时第一次被分到同班，相识的两人。
同班同学、社团伙伴、教师、父母……与周围的关联，自己的成长。
在那个一边被变化与不安追着，一边慌慌张张地向前飞奔的季节之中，稚嫩而又令人眩目的青春期之恋。
