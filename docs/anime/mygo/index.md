# BanG Dream! It's MyGO!!!!!


## 简介

「能一辈子和我搞乐队吗？」

在高一的春季即将结束之时，羽丘女子学园里人人都在参加乐队活动。
入学较晚的爱音，也为了尽快融入班级，急忙寻找乐队成员。
在寻找时，她发现被称为「羽丘的不可思议少女」的灯还没有参加乐队，
于是爱音不由得向她发出邀请…

我们的<音乐(呐喊)>，伤痕累累而充满狼狈。
迷茫也无所谓，迷茫也要前进。

- 脚本：晴日たに、后藤绿、绫奈由仁子、小川瞳、和场明子
- 分镜：林翔子、日高初美
