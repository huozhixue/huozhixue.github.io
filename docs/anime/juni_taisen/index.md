# 十二大战


## 简介

每十二年举办一次的，第十二回“十二大战”。
持有干支之名的十二名狞猛战士，以自己的性命与灵魂为赌注的战斗。
参加者是『子』『丑』『寅』『卯』『辰』『巳』『午』『未』『申』『酉』『戌』『亥』的十二位异样的战士们。

摘得这场战争的胜果的人，任何愿望都可以成真，但数额只有一个。
无论如何都想要成真的、那仅此一个的愿望——

最后活下来的会是谁？在谋策与杀戮的战场漩涡之中，流下的是谁的泪水？
震撼灵魂的盛大战斗，即将开战。

**制作人员：**
- 原作：西尾维新
- 导演：细田直人
- 脚本：高桥龙也、高木登、山田哲弥、村井贞之
- 分镜：牧原亮太郎、原博
