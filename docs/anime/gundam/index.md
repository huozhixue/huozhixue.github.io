# 机动战士高达


## 简介

U.C.（宇宙世纪）0079年，人类向宇宙发展并建立了多个巨大的卫星，让人类能够居住在宇宙中。卫星的居民们不堪忍受地球联邦政府对其剥削和压迫，遂宣布独立，更组建军队与地球联邦政府决一雌雄。在战争中双方先后开发了机动战士并投入战场。NEWTYPE少年阿姆罗·雷无意中驾驶上联邦军新开发的机动战士高达，卷入后来被称为“一年战争”中，并与其宿敌夏亚·阿兹纳布尔相遇。

**制作人员：**
- 原作：富野由悠季、矢立肇
- 脚本：星山博之、荒木芳久、松崎健一、山本优
- 分镜：贞光绅也、藤原良二
