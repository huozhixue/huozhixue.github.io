# 苍之茧


> <u>**[苍之茧](https://bgm.tv/subject/1898)**</u>，又名：ペイル・コクーン、记忆裂痕、PALE COCOON

## 简介

那是巨大的废墟世界。

海和大地消失身姿，过去的风景从被发掘废墟的『记录』中存在着。

发掘与复原记录，从那里分析过去的世界。

一手承担这些的工作的设施『记录发掘局』。

主角面对所有记录，过去的世界历史被埋没着.....
