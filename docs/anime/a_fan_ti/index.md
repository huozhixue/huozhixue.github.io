# 阿凡提的故事


## 简介

阿凡提布偶戏系列美术片于1979～1989年上海美术电影制片厂拍摄。根据流传于新疆民间的《阿凡提故事》改编。第一部影片成功后，继续拍成多集系列片。

影片曾获1979年文化部优秀美术片奖；1980年第三届中国电影“百花奖”最佳美术片奖，全国少数民族题材电影“腾龙奖”美术片一等奖；1991年获美国芝加哥国际儿童电影节一等奖。




**制作人员：**
- 导演：靳夕、曲建方、蔡渊兰、金芳玲、蔡渊澜、刘蕙仪
- 脚本：凌纾
- 人物设定：孙大衡
