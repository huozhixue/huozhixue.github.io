# 镜花缘


> <u>**[镜花缘](http://bgm.tv/subject/111317)**</u>

## 简介


唐武皇时期，秀才唐敖科举失意，心怀抑郁。在妻兄林之洋的邀请下，他随船队出海经商，四海巡游。造访了君子国、女儿国、大人国、两面国、无肠国、穿胸国等民俗各异的国家，一路发生许多刺激和有趣的事情。

改编自清小说家李汝珍的同名作品。影片第4集《两面国》获1991年度广播电影电视部优秀影片奖和1993年第二届中国影视动画展播荣誉奖。
