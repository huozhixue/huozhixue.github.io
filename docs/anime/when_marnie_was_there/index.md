# 回忆中的玛妮


## 简介

海边村庄的老夫妇收养了一个内向的女主人公安娜。少女安娜内心封闭，终日在村中孤独的生活。某一天安娜遇到了神秘少女玛妮，并与她成为了好朋友。然而村中除了安娜，谁都不知道玛妮的存在。


**制作人员：**
- 原作：Joan Gale Robinson
- 导演：米林宏昌
- 脚本：安藤雅司、丹羽圭子
- 音乐：村松崇继
- 美术监督：种田阳平
- 作画监督：稻村武志
