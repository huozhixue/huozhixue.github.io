# 异国迷宫的十字路口


## 简介

原作是武田日向在月刊《Dragon AGE》连载的漫画作品。
19世纪后半、是西欧正好流行日本文化的时代。一位单身前往法国的日本少女汤音，在巴黎街上的工艺店“国王的招牌店（アンセーニュ・ド・ロア）”工作著。本故事即是描述她的成长，与年轻老板的克洛德的往来的巴黎滞留记。

**制作人员：**
- 原作：武田日向
- 导演：安田贤司
- 脚本：佐藤顺一、池田真美子
- 分镜：岛津裕行、后藤圭二、布施木一喜
