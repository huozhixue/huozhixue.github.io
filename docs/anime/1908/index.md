# 动画的世纪·100 部作品

> **2006年法国昂西评选的100部短篇动画作品**
## 简介

从 1908 第一部抽象动画 [幻影集](https://bgm.tv/subject/80198) 到 2004 奥斯卡最佳动画短片 [拯救大师雷恩](https://bgm.tv/subject/111877)

幻影集 是电影史上世界上第一部抽象动画，无声黑白，2006年法国安锡（Annecy）国际动画电影节上，被评为“动画的世纪•100部作品”第二名。

本片无情节无主题，不到两分钟的影像里，导演émile Cohl用再简单不过的线条为观众做了多次变形：人变为酒瓶、酒瓶变为花朵、大象变为房子等等，今天看来，想象力也美得让人叫奇。

**制作人员：**
- 导演：[Émile Cohl](https://bgm.tv/person/33229)
