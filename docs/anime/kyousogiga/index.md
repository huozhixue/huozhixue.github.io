# 京骚戏画


## 简介

在与外界隔绝，时间静止的镜都内，主角筝和不同势力的代表，为了寻找共同的目标——黑兔古都，而每天上演着奇幻缤纷的格斗。众人对自身过去的执著，以及对特殊现象的好奇，渐渐弄清世界的所有现象的前因后果。

**制作人员：**
- 原作：东堂泉
- 导演：松本理惠
- 脚本：丸尾未步、中濑理香
- 分镜：角铜博之、志水淳儿
