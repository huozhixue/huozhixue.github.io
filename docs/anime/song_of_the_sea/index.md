# 海洋之歌


## 简介

故事背景设置在了现代社会，是有关于一个塞尔奇小精灵的民间传说，她是最后一个拥有着海豹外貌却能化为人形的生物，尝试着想要重新回到大海里——那是一个童话世界即将终结的时间点，所有只存在于民间传说中的生物全部在现代社会的影响下，愈加地堕落、腐败，他们根本就不想回去，可小精灵的歌声却是让他们回归的惟一办法……
一个有点悲伤的故事，却以一种非常美好的方式呈现出来。

**制作人员：**
- 导演：汤姆·摩尔
- 音乐：布鲁诺·库莱斯
- 动画制作：卡通沙龙
