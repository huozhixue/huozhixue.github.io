# 悠哉日常大王


## 简介

作品舞台是一个挂著“牛经过注意”的看板，公车不等上5个小时不会来的乡下农村，而主角一条萤因父亲工作调职的关系从东京迁居此地，故事即是描写萤与越谷姐妹及宫内姐妹渡过的日常光景。

**制作人员：**
- 原作：あっと
- 导演：川面真也
- 脚本：吉田玲子、山田由香、志茂文彦
- 分镜：岩崎良明、锦织博、小柴纯弥
