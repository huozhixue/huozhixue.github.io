# 我为歌狂


## 简介

《我为歌狂》是一部校园青春偶像小说。反映校园生活及流行音乐、流行乐队、流行歌手的喜怒哀乐；表现青年生活中所欠缺而又内心热切渴望追求的悲欢离合。小说中的主人公是南华高中的一群帅哥靓女，故事围绕他们组建男生乐队"OPEN”和女生乐队"HAPPY”展开，校园生活、流行音乐、偶像崇拜、少男少女的恋情、学生与老师的冲突……小说与同名电视卡通片于2001年8月1日同步亮相。



**制作人员：**
- 导演：胡依红
- 脚本：倪明
- 音乐：彭程、金复载
- 原画：王昕（洛水花原导演）
- 剪辑：龚黎民、强小柏
