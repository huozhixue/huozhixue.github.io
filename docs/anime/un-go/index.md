# UN-GO


> <u>**[UN-GO](https://bgm.tv/subject/18636)**</u>，又名：因果日记、安吾、アンゴ

## 简介

以日本著名文学家坂口安吾所作《明治开化安吾捕物帖》为蓝本，将这部时代剧作品放置在近未来舞台大胆重现的原创作品，讲述侦探二人组：被称为最后的名侦探的结城新十郎，以及他的搭档——不可思议的美少年因果的故事。
