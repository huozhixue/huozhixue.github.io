# 玉子爱情故事


## 简介

时已入春，已经进入高中三年级的北白川玉子仍然总是想着打糕的事情。
春天傍晚的放学路上，玉子好友四人组谈论着关于将来出路的事情。大家虽然对此感到不安，但似乎对将来的事情都有好好地考虑过。玉子也无意中说出自己将来想要继承家业。
同时，住在玉子家的对面、一直和她在一起的饼藏，也下定了某种决心。
周围的各种事物都在逐渐改变着，一点一点地动摇着玉子的心。

- 脚本：吉田玲子
- 分镜：石原立也
- 演出：河浪荣作、小川太一
