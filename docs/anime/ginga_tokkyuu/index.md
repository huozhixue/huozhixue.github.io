# 银河特急 银河☆地铁


> <u>**[银河特急 银河☆地铁](https://bgm.tv/subject/545328)**</u>，又名：銀河特急 ミルキー☆サブウェイ、银河特急 Milky✩Subway、Ginga Tokkyuu Milky☆Subway

## 简介

该动画番剧将世界观设定在遥远的太空，讲述一群银河系小人物因违反银河交通道路法被捕，他们被派去义务劳动清扫星际列车——“银河特急”，却意外遇上列车失控，在宇宙中暴走……一场前所未有的宇宙列车奇幻之旅就此开启。
