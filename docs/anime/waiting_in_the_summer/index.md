# 在盛夏等待


## 简介

天空湛蓝，万里无云。
积雨云翻涌，仿佛要将彼岸的山隐藏。
这是我们小镇上常见的风景。
但是，也是无可替代的“盛夏”。
那个男孩有一个模糊的想法“一无所有却想做点什么”，
所以和同伴商量一起拍电影。
就在那时，一个“特别”的女孩来到了这个小镇，
然后将男孩的想法视为“特别”。
男孩的名字是——雾岛海人。
女孩的名字是——贵月一夏。
他们的盛夏开始了。
我们在那个盛夏等待——
