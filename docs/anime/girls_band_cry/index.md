# GIRLS BAND CRY


## 简介

无论是愤怒、喜悦还是悲伤，全都倾注其中。
高中二年级，从学校退学，只身在东京以大学为目标的主人公。被同伴背叛，不知如何是好的少女。被父母抛弃，一个人在大城市里打工维生的女孩子。
虽然这个世界总是背叛我们。虽然什么都不能称心如意。但是，因为我们想要喜欢某个东西。因为我们相信自己的容身之处就在某个地方。
所以，我们歌唱。
