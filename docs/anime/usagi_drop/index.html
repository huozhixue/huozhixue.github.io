简介

30岁的单身爸爸养6岁女儿(非亲生)的温情故事。 由制作过《东之伊甸》和《好想告诉你》的Production I.G担任动画制作。 同年夏天将公映由松山健一主演的三次元电影。

制作人员：

  • 原作：宇仁田由美
  • 导演：龟井干太
  • 脚本：岸本卓
  • 分镜：宇井良和、金子秀一、鹤冈耕次郎、井上锐、满仲劝、森田宏幸
