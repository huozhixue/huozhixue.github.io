# 鬼灭之刃


> <u>**[鬼灭之刃](http://bgm.tv/subject/245665)**</u>，又名：鬼滅の刃、Demon Slayer: Kimetsu no Yaiba、鬼滅の刃 竈門炭治郎 立志編、鬼灭之刃 灶门炭治郎 立志篇

## 简介


大正时期、日本。
卖炭的心地善良的少年·炭治郎，有一天被鬼杀死了家人。
而唯一幸存下来的妹妹祢豆子变成了鬼。
被绝望的现实打垮的炭治郎，为了让妹妹变回人类并讨伐杀害家人的鬼，决心沿着“鬼杀队”的道路前进。
人与鬼交织的悲哀的兄妹的故事，现在开始！
