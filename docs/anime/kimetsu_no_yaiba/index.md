# 鬼灭之刃


## 简介

大正时期、日本。
卖炭的心地善良的少年·炭治郎，有一天被鬼杀死了家人。
而唯一幸存下来的妹妹祢豆子变成了鬼。
被绝望的现实打垮的炭治郎，为了让妹妹变回人类并讨伐杀害家人的鬼，决心沿着“鬼杀队”的道路前进。
人与鬼交织的悲哀的兄妹的故事，现在开始！

**制作人员：**
- 原作：吾峠呼世晴
- 导演：外崎春雄
- 脚本：ufotable
- 分镜：南野纯一、野中卓也、寺尾优一、栖原隆史、竹内将
