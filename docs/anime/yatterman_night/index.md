## 简介

《夜之小双侠》以系列作品中的反派角色”杜伦布贼党“的后裔为主人公。小双侠在打败杜伦布后所建立的王国并非如此美好，相反地却让人民饱受苦难。新生杜伦布的首领继承了多龙芝之名，向小双侠发动了复仇......
