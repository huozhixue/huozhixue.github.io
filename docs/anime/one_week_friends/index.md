# 一周的朋友。


## 简介

「但是，我…关于朋友的记忆，一周就会消失呢—」一直都是一个人的同班同学藤宫香织所述的事实。主人公长谷祐树，即使是这样也想和香织成为朋友。一周的记忆开始又结束，两个人不断成为朋友。一次又一次—。对于你，我永远都是你的朋友。然后，总有一天能告诉你，我是你的朋友。
