# 电波女与青春男


## 简介

故事的主角是将“青春点数”分配到自己的行动和相关的事情上、有时欢喜有时忧愁的少年丹羽真，及把被子卷在身上、自称外星人的“电波女”（却是美少女）的藤和艾莉欧（藤和エリオ）。作品以这二人为中心，展开了有点奇怪、却又非常清爽的青春故事。

**制作人员：**
- 原作：入间人间
- 脚本：绫奈由仁子
- 分镜：平川哲生、添田和弘、佐山圣子、杉山延宽、板村智幸、笹木信作、八濑佑树
