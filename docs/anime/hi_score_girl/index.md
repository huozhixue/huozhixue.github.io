# 高分少女


## 简介

以1990年代为背景，描述热衷于格斗游戏的小学生“矢口春雄”，某次在电玩店与女主角“大野晶”因互相较量
《街头霸王2》而认识。故事中除了街机文化之外，也有恋爱要素。由于身份上的差距，毫无交集的春雄和晶，偶然发现有着共同兴趣，
原本只是互相切磋技术的好对手，不知不觉间，两人的关系开始有了变化……

**制作人员：**
- 原作：押切莲介
- 导演：山川吉树
- 脚本：浦畑达彦、石野敦夫、富田赖子
- 分镜：松井仁之、佐山圣子
- 演出：樱美胜志
