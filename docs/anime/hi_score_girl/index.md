去和比我更强的GIRL相见——。
“多边形”？那是啥？能吃吗？就在这种曾是2D全盛期的古早格斗游戏热潮到来的1991年。
在不良、阿宅、上班族蔓延的市郊游戏中心，她凛然地坐在那里——。
通过主人公春雄描绘的“90年代街机恋爱喜剧”！
