# 齐木楠雄的灾难


## 简介

高中生·齐木楠雄是超能力者。心灵感应、念动力、透视、预知、瞬间移动、千里眼等，不论任何事情都自由自在。但这任谁都羡慕不已的最强能力，实际上对于本人而言是引来灾难的不幸元凶。因此，他在别人面前封印了超能力，以不起眼、不和他人有所牵扯为信条，默默无闻地过日子。但不知为何，他的身边总是聚集着不可思议的人类（生物），风暴般的灾难接连不断地降临！

**制作人员：**
- 原作：麻生周一
- 导演：樱井弘明
- 脚本：横谷昌宏、横手美智子、金杉弘子、杉原研二
- 分镜：赤城博昭、名村英敏、梶井濑贺
