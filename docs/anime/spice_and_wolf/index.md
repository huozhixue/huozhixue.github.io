# 狼与香辛料


## 简介

到处旅行靠贩卖一些小商品为生的商人罗兰斯，从因为收获祭而沸腾的帕斯洛耶村回来后却发现自己的运货马车中貌似有什么东西在里面，罗兰斯把麦束拨开一看，里面却睡着一只长有狼耳和狼尾巴的少女。这位少女自称是“掌控丰收的贤狼——赫萝”，靠麦子为生的她如果脖子上挂的帕斯洛耶麦子遗失了就会死。赫萝死赖着劳伦斯希望他能够带她回到遥远的北方故乡，于是，狼女与商人“完全没有剑与魔法的”旅行由此展开……
