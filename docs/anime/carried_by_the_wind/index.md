# 随风舞月影兰


## 简介

本片讲述了一位武艺高强但又嗜酒如命的女武士（月影兰）与一位使用猫铁拳格斗术的小女孩（美安奈）相遇并一起游历于各处的故事。两人一路上行侠仗义，但又不时地闹出点笑话……

**制作人员：**
- 原作：大地丙太郎
- 脚本：高桥良辅、佐藤龙雄、秦义人、横手美智子、池田真美子
- 分镜：政木伸一
