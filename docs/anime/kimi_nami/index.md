# 若能与你共乘海浪之上


## 简介

向水日奈子因考上大学而搬来海边城镇居住，她非常喜欢冲浪，觉得自己只要站在海浪之上就无所畏惧。但其实，她是一个对自己未来没有什么自信的人。因为某场火灾，日奈子和消防员雏罂粟港相遇。通过一起冲浪等经历，日奈子开始被总是为了别人着想的港所吸引。而港的心中，也萌发了对日奈子的特殊情感。
