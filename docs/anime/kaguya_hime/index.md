# 辉夜姬物语


## 简介

《辉夜姬物语》（日语：かぐや姫の物語）是日本吉卜力工作室的动画导演高畑勋以日本古典文学《竹取物语》为蓝本所改编的动画电影。かぐや姬，汉译有不同写法，如“辉夜姬”、“赫映姬”、“竹取公主”等等。但据日本语言学家山田孝雄考证，此词源出汉语“赫奕”一词。

**制作人员：**
- 导演：高畑勋
- 脚本：坂口理子
- 分镜：桥本晋治、佐藤雅子、笹木信作、百濑义行
- 音乐：久石让
- 人物设定：田边修
