# 妄想学生会


## 简介

故事的舞台——私立樱才学园，原本是传统的女子高中，但因少子化问题影响，不得不改制为男女合校。作为改制后第一年入学的28名男性新生之一的津田隆利，偶然之下被拉入了学校的学生会。在外人看来，学生会是美女如云的天堂，但是津田身边的三位学生会女干部性格都非常独特，津田开始了他每天学生会的吐槽生活……

**制作人员：**
- 原作：氏家ト全
- 导演：金泽洪充
- 脚本：小山知子、中村诚
- 分镜：博史池畠、安川胜
- 演出：渡边哲哉
