# 伯纳德小姐说。


## 简介

“伯纳德小姐”町田佐和子与爱读书的朋友们在图书馆展开的文学日常。
对古今东西各种书籍的爱，以及充满“确实是读书家呢”这类细节的“名著礼赞”搞笑剧！
