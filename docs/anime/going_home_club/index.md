# 归宅部活动记录


> <u>**[归宅部活动记录](https://bgm.tv/subject/75702)**</u>，又名：帰宅部活動記録、Chronicles of the Going Home Club

## 简介

作品的舞台是以“尽可能快乐地度过放学后的时光”为主导思想的“归宅部”，故事描绘出其不意地进入了归宅部的女主角安藤夏希与部长道明寺樱等部员谱写的异乎常人的日常。
