# 花田少年史


## 简介

主人翁花田一路於某年發生一次車禍意外，在昏迷中夢見一位名叫小雪的美少女，後來一路被這個鬼魂嚇醒，也開始有看得見鬼魂的超能力，許多鬼魂接近一路，希望他能幫他們完成在人世時尚未完成的願望。


**制作人员：**
- 原作：一色真人
- 导演：小岛正幸
- 脚本：高屋敷英夫、金春智子、米村正二、望月武、藤田伸三
- 分镜：兼森义则、望月智充
