# 天体的秩序


## 简介

某日在雾弥湖的上空中出现了谜样的圆盘，一度造成全世界的混乱与恐慌，但飞碟却只停留在空中，并没有进一步的行动。在时间流逝中人们渐渐对这件事失去了兴趣......就在两名少女相遇后，静止的时间再次流动，让众人有所改变的全新故事展开，中学生活最后那一年，将拥有一生无法忘怀的回忆。

- 演出：铃木拓磨（动画人）、熨斗谷充孝、室谷靖、佐土原武之
