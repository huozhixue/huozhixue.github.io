# 百变孙悟空


## 简介

韩国经典动画片，共五季54集。
手拿双节棍脚踩滑板的孙悟空，开越野车的唐僧，拿大炮的猪八戒，还有使用小锤的沙僧，共同演绎现代版《西游记》的故事。

**制作人员：**
