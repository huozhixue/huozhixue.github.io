# 灰与幻想的格林姆迦尔


## 简介

我们为什么要这么做……？

哈尔希洛回过神来，才发现自己身处在黑暗当中，他完全不知道自己人在何处，也不明白这个地方是哪里。

他的身边有一群和他一样失去记忆，只记得自己名字的男女；而离开了地底后，等待着众人的是一个「宛如游戏」的世界。

为求生存，哈尔希洛与自己有着相同境遇的伙伴们组成队伍、学习技能，以义勇兵见习者的身份踏入了这个世界--「格林姆迦尔」。

没有人知道未来会遇见什么……这一切，就是从灰烬之中所诞生的冒险谭。
