# 东京残响


> <u>**[东京残响](http://bgm.tv/subject/100443)**</u>，又名：残響のテロル、Terror in Resonance、恐怖残响

## 简介


某个平凡的夏日。
突然，东京遭到了大规模炸弹恐怖袭击。
这个导致日本从平稳的睡梦中惊醒的事件的犯人，仅仅是两个少年。
自称为“斯芬克斯”的犯人们，将把全日本卷中一场壮大的游戏。
