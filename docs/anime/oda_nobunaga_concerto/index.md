# 信长协奏曲


## 简介

讨厌念书，对日本史毫无兴趣的高中生三郎，某一天突然时光穿越到了战国时代。他在那里见到体弱多病的织田信长本人，而信长的相貌竟然与三郎一模一样。信长拜托三郎代替体弱多病自己，以织田信长的身分活下去…！？用全新的手法诠释出来的织田信长，叫人出乎意料之外却又莫名地符合信长形象的三郎。对历史发展可说是毫无头绪的他，在战国时代会如何生存下去？

**制作人员：**
- 原作：石井步
- 导演：富士川佑辅
- 脚本：高桥奈津子
- 分镜：森田纯平
- 音乐：横山克
