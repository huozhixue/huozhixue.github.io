# 真女神转生 恶魔之子


## 简介

竹内启雄魔鬼混血儿是指魔鬼与人类之间的孩子，因此，一到关键时刻，额头就会出现五角星，并能发挥神秘魔力。那是一种传说中的力量，足以毁灭世界。"刹那"与"未来"之所以能够驱使"德比拉泽" 和"比内高"与魔鬼敌人战斗交涉，召唤魔鬼同伴，就是因为他们是魔鬼混血儿。魔鬼混血儿"甲斐 刹那"和"要 未来"有何宿命？所有谜底都在魔界。

- 色彩设计：加藤良高
