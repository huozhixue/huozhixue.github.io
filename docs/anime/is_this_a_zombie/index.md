# 这个是僵尸吗？

## 简介


平凡的高中生相川步，在莫名其妙的因缘下被卷入震惊社会的连环杀人案身亡，但他竟然死而复生了。原来在事件发生之前与步偶然邂逅的死灵法师“优”，将他变成了不死身而复活。再加上来自魔法世界的少女战士春奈，与另一个吸血忍者少女瑟拉芬，甚至演变成步跟她们全住在一起的局面。
