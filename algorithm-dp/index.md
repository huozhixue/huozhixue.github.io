# 力扣总结 常见算法（六）：动态规划


动态规划是一种特殊的分治法，适用于有重叠子问题的场景。

存在大量重叠的子问题时，普通的递归写法会重复计算这些子问题，浪费大量时间。
动态规划会保存所有子问题的结果，避免重复计算，将时间复杂度减少为多项式级别。

动态规划也有递归和非递归两种形式。递归形式的动态规划也叫记忆化递归，非递归形式即是递推。

python 在递归函数上加一个 @lru_cache(None) 的装饰器即可实现记忆化递归。
非递归形式则根据参数量个数 k 创建一个 k 维的 list 用来保存子问题的结果。
