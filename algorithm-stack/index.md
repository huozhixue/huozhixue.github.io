# 力扣总结 数据结构（五）：栈和队列


栈是一个后入先出的数据结构，插入和删除都在末尾进行。python 中一般直接用 list 表示。

队列是一个先入先出的数据结构，插入在末尾，删除在开头。python 中一般直接用内置库 deque 表示。

额外地，dfs 常借助栈实现，bfs 常借助队列实现，这里先不涉及。
