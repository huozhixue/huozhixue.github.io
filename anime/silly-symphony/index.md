# 迪士尼糊涂交响曲


本合集是迪斯尼在 1929-1939 十年期间推出的一系列动画短片

最为出名的有：

- 01：1929，骷髅舞，系列首部
- 29：1932，花与树，奥斯卡首部最佳短片
- 36：1933，三只小猪，奥斯卡最佳短片
- 49：1934，龟兔赛跑，奥斯卡最佳短片
- 52：1935，水孩子
- 56：1935，三只孤儿猫，奥斯卡最佳短片
- 63：1936，乡巴佬，奥斯卡最佳短片
- 68：1937，老磨坊，奥斯卡最佳短片
- 75：1939，丑小鸭，奥斯卡最佳短片
