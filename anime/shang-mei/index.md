# 上美早期动画短片


本合集收录上美早期动画短片（代表性的），从 1950《谢谢小花猫》到 1965《红领巾》

最出名的有：

- p8，1955.神笔，
- p11，1955.乌鸦为什么是黑的，中国第一部彩色单线平涂动画
- p16，1956.骄傲的将军，第一部京剧风格动画
- p29，1958.猪八戒吃西瓜，中国第一部剪纸动画
- p48，1959.雕龙记
- p49，1960.小蝌蚪找妈妈，第一部水墨动画
- p56，1960.聪明的鸭子，中国第一部折纸动画
- p60，1962.没头脑和不高兴
- p67，1963.牧笛
- p73，1964.半夜鸡叫
