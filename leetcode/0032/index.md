# 0032：最长有效括号（★★★）


## 题目

给你一个只包含 '(' 和 ')' 的字符串，找出最长有效（格式正确且连续）括号子串的长度。


<!--more--> 

示例 1：

	输入：s = "(()"
	输出：2
	解释：最长有效括号子串是 "()"
	
示例 2：

	输入：s = ")()())"
	输出：4
	解释：最长有效括号子串是 "()()"

示例 3：

	输入：s = ""
	输出：0

 
## 分析 

先想想人是怎么做的，从左往右看，每次遇到右括号就找其对应的左括号，两个括号中间就是有效的。
然后如果前面相邻的一段也是有效的，就合并在一起，这样就得到该右括号为结尾的最长有效子串。

具体实现时，可以用 d[i] 代表位置 i 的右括号结尾的最长有效括号的起点，对应的长度即为 i-d[i]+1。。
计算 d[i] 时，先找到该右括号对应的左括号位置 j，如果位置 j-1 也是右括号，那么 d[i] = d[j-1]，否则 d[i] = j。


## 解答

```python
def longestValidParentheses(self, s: str) -> int:
	res, stack, d = 0, [], {}
	for i, char in enumerate(s):
		if char == '(':
			stack.append(i)
		elif stack:
			j = stack.pop()
			d[i] = d.get(j-1, j)
			res = max(res, i+1-d[i])
	return res
```

时间复杂度 O(N)，52 ms
