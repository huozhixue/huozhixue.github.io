# 0031：下一个排列（★★）


## 题目

实现获取 下一个排列 的函数，算法需要将给定数字序列重新排列成字典序中下一个更大的排列。

如果不存在下一个更大的排列，则将数字重新排列成最小的排列（即升序排列）。

必须 原地 修改，只允许使用额外常数空间。


<!--more--> 

示例 1：

	输入：nums = [1,2,3]
	输出：[1,3,2]
	
示例 2：

	输入：nums = [3,2,1]
	输出：[1,2,3]
	
示例 3：

	输入：nums = [1,1,5]
	输出：[1,5,1]
	
示例 4：

	输入：nums = [1]
	输出：[1]


 
## 分析 

因为必须原地修改，所以只能找规律了，可以先写出 [1, 2, 3, 4] 按顺序的所有排列来观察：

	1 2 3 4
	1 2 4 3
	1 3 2 4
	1 3 4 2
	1 4 2 3
	……

发现当后缀 nums[i:] 能够增大时，显然不需要再考虑 nums[:i] 的部分了。
所以反序找到第一个 i 使得 nums[i:] 能够增大，将问题缩小为找 nums[i:] 的下一个排列。

此时必然有 nums[i+1:] 是逆序排列（所以不能增大），而 nums[i] < nums[i+1]。
下一个排列其实就是增大得尽量小，因此在 nums[i+1:] 中找到最后一个比 nums[i] 的数大的数，与 nums[i] 交换。
再将 nums[i+1:] 正向排序使其最小即可。

例如 [1, 3, 4, 2]，反向找到第一个能增大的后缀 [3, 4, 2]，将 3 与 4 交换，再把 [3, 2] 排序即可。


## 解答

```python
def nextPermutation(self, nums: List[int]) -> None:
	n = len(nums)
	i = n-2
	while i >= 0 and nums[i] >= nums[i+1]:
		i -= 1
	if i >= 0:
		j = n-1
		while nums[j] <= nums[i]:
			j -= 1
		nums[i], nums[j] = nums[j], nums[i]
	l, r = i+1, n-1
	while l < r:
		nums[l], nums[r] = nums[r], nums[l]
		l += 1
		r -= 1
```
