# 0390：消除游戏（★★）


第 2 场周赛第 2 题

## 题目

给定一个从1 到 n 排序的整数列表。
首先，从左到右，从第一个数字开始，每隔一个数字进行删除，直到列表的末尾。
第二步，在剩下的数字中，从右到左，从倒数第一个数字开始，每隔一个数字进行删除，直到列表开头。
我们不断重复这两步，从左到右和从右到左交替进行，直到只剩下一个数字。
返回长度为 n 的列表中，最后剩下的数字。

 <!--more--> 
 
示例：

	输入:
	n = 9,
	1 2 3 4 5 6 7 8 9
	2 4 6 8
	2 6
	6

	输出:
	6


## 分析

显然用长度为 n 的任意数组玩游戏，最终剩下的位置是一定的。
为了方便，令 dp[i] 代表用长度为 i 的数组 A 玩游戏的结果是第 dp[i] 个数字。

	第一轮后，剩下长度为 i//2 的数组 A[1::2]，然后转为递归子问题用 B=A[1::2][::-1] 玩游戏。
	
	故最终数字在 B 中是第 dp[i//2] 个，然后可以推得最终数字在 A[1::2] 中是第 len(B)+1-dp[i//2] 个。
	
	故最终数字在 A 中是第 2*(len(B)+1-dp[i//2]) 个。
	
	即得到递推式：	dp[i] = 2*(i//2+1-dp[i//2])
	
本题中第 dp[n] 个数字就等于 dp[n]，返回即可。

## 解答

```python
def lastRemaining(self, n: int) -> int:
	return 2*(n//2+1-self.lastRemaining(n//2)) if n > 1 else 1
```
