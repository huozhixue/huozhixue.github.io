# 0172：阶乘后的零（★）


## 题目

给定一个整数 n，返回 n! 结果尾数中零的数量。

说明: 你算法的时间复杂度应为 O(log n) 。
	
 <!--more--> 

示例 1:

	输入: 3
	输出: 0
	解释: 3! = 6, 尾数中没有零。

示例 2:

	输入: 5
	输出: 1
	解释: 5! = 120, 尾数中有 1 个零.

## 分析

显然结尾零的个数等于 n! 的质因数中所有 (2, 5) 对的个数。显然 5 更少，因此结果就等于因子 5 的个数。

从 1 到 n ，含有 1 个因子 5 的个数是 n // 5，含有 2 个因子 5 的个数是 n // 25，因此结果为：

	n // 5 + n // 25 + n // 125 + ...
 
## 解答

```python
def trailingZeroes(self, n: int) -> int:
	res = 0
	while n > 0:
		n //= 5
		res += n
	return res
```
